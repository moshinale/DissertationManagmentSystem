# Dissertation Management System

## 📌 Overview

The Dissertation Management System is a full-stack web application designed to streamline the process of managing student dissertations and faculty approvals. The system provides an efficient platform for students, supervisors, and administrators to collaborate, track progress, and manage submissions in real time.

This project focuses on improving workflow efficiency, reducing manual processes, and enhancing communication between all stakeholders involved in dissertation management.

---

## 🚀 Features

### 👨‍🎓 Student Module

* Submit dissertation proposals
* Upload project documents
* Track approval status in real-time
* Receive feedback from supervisors

### 👨‍🏫 Supervisor Module

* Review and approve/reject submissions
* Provide feedback and suggestions
* Monitor assigned students’ progress

### 🛠️ Admin Dashboard

* Manage students and supervisors
* Assign supervisors to students
* Track overall dissertation progress
* View system analytics and reports

### ⚡ Core Functionalities

* Real-time database integration using Firebase
* Secure authentication system
* Role-based access control (Admin, Supervisor, Student)
* Interactive and user-friendly dashboard
* Efficient data management and tracking

## 📂 Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/dissertation-management-system.git
   ```

2. Navigate to project folder:

   ```bash
   cd dissertation-management-system
   ```

3. Open `index.html` in your browser
   OR use Live Server in VS Code

4. Configure Firebase:

   * Create a Firebase project
   * Add your config file in the project
   * Enable Authentication & Firestore
